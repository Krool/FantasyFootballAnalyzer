# Archived docs

Point-in-time documents kept for history. They are not maintained and may
describe code that has since changed. For current guidance see `CLAUDE.md`
and the live docs under `docs/`.

| File | What it was | Superseded by |
|---|---|---|
| `IMPLEMENTATION_PLAN.md` | The original 8-phase feature plan (PAR parity, luck, rivalries, awards, history, player journey, polish). All phases shipped. | The features now live in `src/`; `docs/FANTASY_FOOTBALL.md` and `docs/API_REFERENCE.md` document the result. |
| `FANTASY_FOOTBALL_RULES.md` | An early rules primer focused on the "games started = lineup starts, not NFL games" distinction. | `docs/FANTASY_FOOTBALL.md` (domain reference, including the games-started note). |

The dated review snapshots that remain under `docs/`
(`PROJECT_REVIEW_2026-06.md`, `YAHOO_DATA_COMPARISON_2026-06.md`) are also
historical, but stay in place because their filenames carry their own date.
